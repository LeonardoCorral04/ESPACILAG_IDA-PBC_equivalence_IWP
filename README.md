# ESPACILAG_IDA-PBC_equivalence_IWP
Case study of which ESPACILAG methodology conditions design a controller equivalent to a IDA-PBC one for a Inertia Wheel Pendulum. 
Exercise done on Laboratorio de Vehículos Autónomos at Instituto Tecnológico de La Paz during my undergraduate community service. 

Order of MATLAB files:
  1. IDA_PBC_IWP_nbt
  2. ESPACILAG_IWP_IDA_PBC_qp_nbt
  3. ESPACILAG_speed_reg_IWP_s0

All details and sources are covered in m files.
